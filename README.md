## 📌 Project Overview

This is a **modern, responsive portfolio website** built using **HTML, CSS, JavaScript, and Bootstrap**.
It is designed to showcase personal projects, skills, and professional information with smooth animations and a clean UI.

---

## ✨ Features

* 📱 Fully responsive design (Mobile + Desktop)
* 🎨 Modern and clean user interface
* ⚡ Smooth animations and transitions
* 🧩 Project showcase section
* 📬 Contact form included
* 🔧 Easy to customize and extend

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
